# Deprecated deploy artifacts

Files here are kept for the audit trail but should NOT be followed.

## `install-memory-stack-2026-04-20-DEPRECATED.md`

Early draft of a memory stack install runbook written before the live install. Contains invented OpenClaw config keys (`memory.embedding`, `memory.contextEngine`, etc.) that don't exist in OpenClaw 2026.4.15's schema. Some of its commands are real; many aren't. **Use `deploy/install-flyn.sh` instead.**

See `POSTMORTEM-2026-04-21.md` at the repo root for why the current shape of the deploy script is what it is.
